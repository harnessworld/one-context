# 短视频 / 精华剪辑版（约 60–90 秒）

可直接单独录音或剪进全长成片前置钩子。**数字与引语须在发布前按 `spec.md` 核查。**

---

**小夏**：Claude Code 工程师 Thariq 在 X 上说：自己彻底不用 Markdown 了——帖子短时间冲上超高浏览，程序员吵翻了：Markdown 要被 HTML 取代？

**阿哲**：冷静听结论：他不是否定 Markdown，而是说 AI 代写时代，「手写友好」不再是核心优势；长文档难读、复杂信息展不开、交互弱，才是痛点。

**小夏**：HTML 强在表达力、网页级阅读、链式分享，还有双向控件能把参数同步回 AI——关键是协作闭环。

**阿哲**：他还强调 Claude Code 能读仓库、Git、接 MCP，生成页面更「贴地」。代价也有：生成更慢、diff 更吵，Token 要自己掂量。

**小夏**：最后一句话送你：Markdown 不会死，Simple stays simple；要复杂协作与交互，再考虑 HTML。选对场景，比追风口重要。
