layer 3-4 · knowledge packaging

第 3-4 层：Skills + Plugins

按需加载专业知识 + 把部落知识打包 防止丢失。

on-demand skills + plugin packaging

Skills 按需加载

几十种任务 ≠ 全部塞进每个会话；需要时才展开。

Plugins 解决部落知识

好配置随员工流失而消失 → 打包后统一分发。

layer 5 · precise navigation

第 5 层：LSP 精准导航

语言服务器协议 — grep 大海捞针 vs 符号精确导航。

grep chaos vs LSP precision

无 LSP 的困境

只能文本匹配，context window 被无关结果快速消耗。

前置投资

部署前全公司铺设 LSP 集成 — 不是开箱即用，需手动配置。

layer 6-7 · advanced integration

第 6-7 层：MCP 服务器 + 子 Agent

连接内部工具 + 探索与编辑分离。

MCP connectors + sub-agent isolation

MCP：段位最高的团队

连接内部数据分析平台，业务分析师不用切换工具。

子 Agent：关键洞察

同一会话又探索又编辑 → context 撑爆；分离后只返结果。
