# 幻灯 ↔ 字幕（`sub.srt`）对照说明

口播音轨：`production/media/podcast-action0.wav`  
字幕：`production/subtitles/sub.srt`（Whisper 时间轴）

## 能否自动对齐成片？

**当前不能。** 目录里没有 `wav-durations.json`，`html-video-from-slides` 的 **wav 模式**无法按页翻页；若用 **wav-auto**，依赖 Whisper 匹配幻灯 **可见文字**，本稿字少、播客为改写叙述，**对齐风险高**。

要对齐成片，需要在本目录补 **`wav-durations.json`**（或跑 `srt-map` 手工指定 `SRT 条目区间 → 页码`），并用 `timing-check` 自查翻页是否落在过渡句上。

## 第 3 页（Agenda）与音频的关系

**口播里没有「目录页」独白**：播客在讲完钩子与蓝图读法后，很快进入「我们来谈谈四大核心技术拼图的 第一个…」（约 **`00:02:06`** 起）。  
因此 **第 3 页更像剪辑用结构屏**：若要做严丝合缝，常见做法是：

- 在 **`00:02:00`～`00:02:08`** 插 **8～12 秒** 目录镜头；或  
- 剪片时 **跳过第 3 页**，把第 4 页紧接在第 2 页后。
