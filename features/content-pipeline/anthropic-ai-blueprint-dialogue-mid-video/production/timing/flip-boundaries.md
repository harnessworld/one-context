## 特别说明（仍算「确定」、不是模糊带）

1. **Cover 与 Trap 的分界**  
   「我们先来看看 / 热议蓝图 / **每一层对应工程环节** / 不是简单示意图」整段必须在 **Cover**（含蓝图分层视觉）；**#18 标题党** 起才进入 **Trap**。

2. **Agenda（第 3 页）**  
   口播无独立目录句；**126.360 s** = **#55 的结束时间戳**，**1.56 s** 后（**127.920 s**）进第 4 页，与 **#57** 首字对齐。期间 **#56**「我们来谈谈」仍按字幕正常播放、画面为目录页。

3. **第 4 页仅 2.48 s**  
   与口播上「总览一句 → 立刻进 Managed」一致；若你日后要拉长总览，只改 `slideDurationsSec[3]`、**[4]** 并同步改本表第 4、5 行「进入时刻」。

4. **与 `wav-durations.json` 的对应**  
   `slideDurationsSec[i]` = 上表「进入第 i+1 页」与「进入第 i+2 页」之 **差**；最后一项为 **音轨尾 − 进入第 13 页时刻**。
