## 标注约定（重音 / 停顿 / 互动）

录制或剪辑时可按下述符号标注在文案旁（或在下一版正文内联）：

| 符号 | 含义 |
|------|------|
| `**词**` | 重读 / 加重 |
| `/` | 短停顿（约 0.3–0.5s） |
| `//` | 长停顿（约 0.8–1.2s） |
| `【 … 】` | 互动卡点（弹幕提问、定格、字幕引导） |

### 示例格式（占位）

```text
**你敢信吗** / 顶尖工程师 /**2026** 年一行代码没写 // 【弹幕：你能接受吗】
```
