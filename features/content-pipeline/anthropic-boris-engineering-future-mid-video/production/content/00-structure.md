# 话题大纲

对话体口播：A 追问、B 展开，节奏轻松、信息递进。以下为段落与 **预估口播时长**（正常语速，可按剪辑加减）。

## 幻灯翻页（硬规则）

- **单页口播不得短于 20 秒**（以 `slideDurationsSec` 或 SRT 段落边界为准）。
- 若某页会 **不足 20 秒**：**合并**到相邻同话题页，或把该页口播块并入上一/下一屏；**禁止**为凑页数留空镜短切。
- 与 `html-video-from-slides` 的 `wav` 模式校时时：`slideDurationsSec` 每项 **≥20**，且各项之和等于口播 WAV 总时长。

| 段 | 主题 | 核心梗 | 预估时长 |
|----|------|--------|----------|
| 开场 | 钩子：顶尖工程师不写代码 + 150 PR | 反差、红杉/Boris 背书 | ~45–60s |
| 第一问 | AI 编程不等于 Tab 补全 | 完整生成、产品先于模型、「早期难用≠方向错」 | ~60–90s |
| 第二问 | 手机管几百 Agent | 并行密度、Loop、24h 后台自动化 | ~75–95s |
| 第三问 | 团队人人写代码 | 代码作表达工具、跨学科通才 | ~60–80s |
| 第四问 | 印刷机时刻 | 类比、软件更碎更贴近业务、护城河迁移 | ~70–90s |
| 第五问 | 护城河在流程 | 新公司 vs 大公司、任务拆解、MCP | ~75–95s |
| 结尾 | 用 AI 的人取代不用的人 | CTA：拆工作为可循环任务 | ~45–60s |

**全片粗算**：约 **8–12 分钟**（播客向）；若只做短视频，用 `01-script-short.md` 压至约 3 分钟并保留 1–2 个最强钩子。
