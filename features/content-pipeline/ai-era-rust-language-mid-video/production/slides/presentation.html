06 / 07
Why The Compiler Helps

其实核心就是三招

①
所有权 + 生命周期
把「谁能借引用」写进类型系统，降低 UAF / 数据竞争面。
②
代数类型 + Result
错误路径显式化，减少「吞掉异常」的隐形分支。
③
Send / Sync 并发边界
跨线程共享前先过编译器门禁，适合高并发服务骨架。
④
unsafe → FFI 边界
把 C ABI / SIMD 热点封进窄接口，外层仍保持可审计 Rust。
