# 视频结构方案 — 短视频式汇报范式

## 定位

- **时长**: 约 4 分钟（B站/视频号完整版）；另出 2 分钟精简版（抖音）
- **形式**: 双人口播（A/B 对谈），配合关键画面插片
- **平台分发策略**:
  - **抖音**: 2 分钟精简版（砍第三个示范 + 压缩收尾），完播率优先
  - **B站**: 4 分钟完整版，信息密度优先
  - **视频号**: 4 分钟完整版，封面加信任背书
- **角色**:
  - **A（小周）**: 方法论输出担当，从自己翻车经历中提炼出解法的人
  - **B（老林）**: 场景还原 + 质疑担当，负责演绎真实翻车场景、提出反例和追问
- **核心记忆点**: 「时间是稀缺的，注意力更是」

## 结构

| 时间轴 | 段落 | 钩子类型 | 注意力设计 |
|--------|------|----------|-----------|
| 0:00–0:05 | **开场钩子** | 痛感钩 | 具体场景+反常识结果，5 秒内制造「卧槽然后呢」 |
| 0:05–0:35 | **痛点共鸣** | 镜像钩 | 还原真实汇报场景，触发「对对对就是我」；承认老板有时确实在忙 |
| 0:35–1:10 | **范式翻转** | 认知钩 | 翻转前提：汇报的本质是帮老板更快做出更好的决策 |
| 1:10–2:00 | **原则拆解 + 场景示范（融合）** | 干货钩+示范钩 | 2 条核心原则嵌入周报 Before/After，边讲边演，B 每条提出质疑；剩余 2 条原则融入下段 |
| 2:00–2:50 | **场景示范续 + 被打断应对** | 示范钩 | 第 2 个 Before/After（要资源）含「被老板打断」应对演示，原则③④嵌入其中 |
| 2:50–3:10 | **老板真正想听什么** | 认知钩 | 快速补充：风险早暴露、决策点明确、承诺追踪、团队对齐 |
| 3:10–3:40 | **收尾钩子** | 回声钩 | 金句收束 + 评论区互动引导 |

### 抖音 2 分钟版裁剪方案

| 保留段落 | 时间 | 说明 |
|----------|------|------|
| 开场钩子 | 0:00–0:05 | 同完整版 |
| 痛点共鸣（压缩） | 0:05–0:25 | 只保留最强的一个场景 |
| 范式翻转 | 0:25–0:45 | 同完整版 |
| 原则+示范（融合，只留 1 个） | 0:45–1:20 | 2 条核心原则 + 周报 Before/After |
| 被打断应对（精华版） | 1:20–1:40 | 要资源 After + 打断应对演示，作为第二高潮 |
| 收尾 | 1:40–2:00 | 金句 + 评论引导，不做倒数回顾 |

## 节奏图

```
注意力
  ^
  |  *开场         *翻转        *示范高潮
  |  /\    痛点     /\  原则+演示  /\     收尾
  | /  \--共鸣---/  \--融合---/  \---金句--/->
  |/    \       /    \      / \   \    /
  +---------------------------------------- 时间
  0    0:35    1:10   2:00   2:50  3:40
```

节奏原则：每 30–40 秒设一个小高潮（*标记），两个高潮之间允许轻微下沉用于消化，但绝不允许连续 20 秒无信息增量。

## 画面设计

| 段落 | 画面 |
|------|------|
| 开场钩子 | A/B 双人怼脸 + 闪切 20 页 PPT → 3 页的视觉对比，A 掷出痛感场景后停顿 |
| 痛点共鸣 | B 单人还原翻车经历 + 画面插片（PPT 截图、会议室白板、消息列表） |
| 范式翻转 | A 画白板/纸上推导，B 逐条追问确认 |
| 原则+示范融合 | 左右分屏 Before/After，每条原则出字幕卡片，B 提出质疑后 A 回应 |
| 老板视角补充 | 画面切到"老板内心 OS"风格字幕 |
| 收尾钩子 | 回到双人怼脸，与开场构图呼应 + 评论引导字幕 |

## 记忆点设计

- **Slogan**: 「时间是稀缺的，注意力更是」（字幕/封面版）；口播辅助版：「人到了，不代表在听」
- **锚点动作**: A 在开场和收尾都用同一个手势（指点/敲桌）配合 slogan
- **编号锚**: 4 条原则用「①②③④」编号，分散嵌入示范段自然带出，收尾不做倒数回顾，改用金句 + 行动指令收束

## 评论区互动设计

- 翻转段植入：「你们老板开会时什么状态？评论区说说」
- 收尾段植入：「你老板开会是刷手机还是打瞌睡？评论区打 1 或 2」
- 传播引导：「转给你那个做 PPT 做到凌晨三点的同事」

## 封面方案

| 平台 | 封面风格 | 标题 |
|------|---------|------|
| 抖音 | 大字 + 人物表情 | 「老板20页PPT只看了3页——但他批了」 |
| B站 | 信息量型 | 「汇报 = 短视频？4条原则让老板听完你的话」 |
| 视频号 | 信任感型 | 「为什么有些人5分钟讲完，老板当场拍板？」 |
