# Security

Please **do not** file public GitHub issues for undisclosed security vulnerabilities.

Report them privately using **GitHub Security Advisories** for the canonical repository: open **Security → Report a vulnerability** (or the advisories page) on [harnessworld/one-context](https://github.com/harnessworld/one-context).

If you are using a **fork**, use the Security tab on your fork if enabled, or report to the upstream maintainers of the canonical repository.

We will acknowledge receipt and coordinate a fix and disclosure timeline with you.
