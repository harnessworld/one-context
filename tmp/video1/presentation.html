04 / 07
一个 UUID 让成本
暴涨 12 倍
💥 事故原因
沙箱配置中的临时路径包含随机 UUID
→ 每次 API 调用的工具描述都不同
→ Prompt Cache 前缀永远匹配不上
→ 每次全量计算，全价收费
✅ 修复方案
随机 UUID → 内容哈希
设置内容没变 → 路径不变 → 缓存命中
一行代码改动，成本直降 12 倍
修复前 · 每次请求
// 每次启动都是新路径 /tmp/a3f8c2d1-.../settings /tmp/7b2e9f4a-.../settings /tmp/e1d5c8b3-.../settings // ❌ 缓存全部 MISS
修复后 · 内容哈希
// 内容不变 = 路径不变 /tmp/sha256(content)/settings /tmp/sha256(content)/settings /tmp/sha256(content)/settings // ✅ 缓存全部 HIT
