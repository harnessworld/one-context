# 情绪分类标准

## 五类情绪定义

| 类别 | 标识 | 分值 | 典型关键词 | 说明 |
|------|------|------|-----------|------|
| 看多 | 🟢 | +1 | 买入、加仓、满仓、涨停、牛市、抄底、起飞、暴赚、突破、翻倍、低吸 | 看好后市，积极看涨 |
| 中性 | 🟡 | 0 | 观望、等信号、持有不动、不确定、震荡、正常波动 | 客观陈述，无明显倾向 |
| 看空 | 🔴 | -1 | 卖出、清仓、割肉、跌停、空仓、止损、跑路、崩盘、下跌 | 看衰后市，消极看跌 |
| 愤怒 | 😤 | -1 | 垃圾、狗庄、骗我、赔死了、坑爹、爆亏、无耻、作弊、出货、割韭菜 | 情绪化抱怨，痛斥主力/公司 |
| 焦虑 | 😰 | -0.5 | 怕跌、要不要卖、熬不住了、睡不着、焦虑、恐慌、要不要割、怎么办 | 纠结犹豫，对亏损担忧 |

## 评分计算

```
情绪分 = (看多占比×10 + 中性占比×5 + 看空占比×0 + 愤怒占比×0 + 焦虑占比×3) / 100
```

换算为 0-10 分，5.0 为中性。

## 特殊规则

1. **吹票识别**：出现"明天必涨""目标价XX""跟庄""老师推荐"等明显吹票话术，额外-0.5分（出货嫌疑）
2. **骂人=愤怒**：凡带脏字/强烈情绪词，无论内容，一律归愤怒类
3. **真实持仓披露**："满仓梭哈"、"清仓了"比"观望"更反映情绪权重，应加权
4. **帖子阅读量**：阅读量高的帖子情绪权重 ×1.5
