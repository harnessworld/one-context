# 翻页时刻 — 语义契约（模板）

在 **`node cli.js wav`** 或流水线 **Step 11** 之前，把本文件复制为 **`timing/flip-boundaries.md`** 并填完。**`timing-check` 只防「句首陷阱」，不防「话题已在下一页但画面还在上一页」**——这张表是人的语义门控。

## 填写规则

1. **进入第 n 页的音频时刻**：从音轨 `0:00` 起的秒数（与 `wav-durations.json` 前缀和一致）。
2. **锚点**：优先写 **`sub.srt` 条目序号** + 该条 **起始时间码** + **起始几个字**；翻页时刻应对齐「**下一页话题已独立成句**」的起点，而非过渡语起点（见 SKILL.md「翻页边界语义」）。
3. 每一行都要能回答：**听众在这一秒听到的核心命题，与当前页 `data-title`/幻灯主题是否一致**。
