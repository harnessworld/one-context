# Cover Design Skill

独立封面设计规范，确保不同项目封面风格一致。

## 快速开始

1. **选择风格** → `PRESETS.md`（3 种预设）
2. **复制模板** → 选择竖版/横版 HTML
3. **填内容** → 替换【占位符】
4. **截图输出** → `node cli.js cover --project <dir>`

## 文件说明

| 文件 | 用途 |
|------|------|
| `SKILL.md` | 核心规范、画布规格、工作流程 |
| `PRESETS.md` | 3 种风格预设（简约/科技/数据） |
| `ELEMENTS.md` | 可组合的封面组件 |

## 风格选择

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| 简约型 | 大标题 + 微装饰 | 知识分享、教程、访谈 |
| 科技型 | 渐变 + 光晕 + 进化条 | 技术、产品、数据类 |
| 数据型 | 超大数字 + 对比 | 业绩报告、年度总结 |

## 字号规范（手机屏幕友好）

| 元素 | 竖版最小 | 横版最小 |
|------|---------|---------|
| 主标题 | 140px | 100px |
| 副标题 | 60px | 44px |
| 说明文字 | 36px | 28px |
