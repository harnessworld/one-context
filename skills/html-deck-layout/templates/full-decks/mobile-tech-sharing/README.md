# mobile-tech-sharing

## When to use
技术分享/技术演讲专用模板，适合 AI 推理、系统架构、性能优化等技术主题。采用深蓝科技主题，强调数据可视化和架构图示。

## Page breakdown

| # | Title | Layout |
|---|-------|--------|
| 1 | 封面 | Cover (主题+演讲者信息) |
| 2 | 背景与挑战 | Split (左文右痛点列表) |
| 3 | vLLM 架构概览 | Card + Diagram (架构流程图) |
| 4 | KV Cache 优化 | 2x2 Grid (四种优化策略) |
| 5 | 连续批处理 | Process Flow (步骤流程) |
| 6 | 量化与压缩 | Split (左数据右说明) |
| 7 | 性能基准测试 | Stat (核心指标) |
| 8 | 总结与展望 | Thanks + 要点回顾 |

## Customization
- Replace theme: 默认使用 mobile-tech.css，可替换为其他主题
- Replace content: 修改各 slide 中的中文内容及 SVG 图示
- Add diagrams: 在 card-diagram 区域添加自定义 SVG
