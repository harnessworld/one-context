技术架构

vLLM 核心架构

PagedAttention 革命

借鉴操作系统虚拟内存思想，将 KV Cache 划分为固定大小的块(Block)，实现高效的显存管理和内存共享

技术架构 03
核心技术

KV Cache 优化策略

🧩

分块存储

将 KV Cache 拆分为固定大小的块(通常 16 tokens)，非连续存储，通过 Block Table 映射逻辑位置到物理块

🔄

内存复用

相同前缀的序列共享物理块，Copy-on-Write 机制确保独立性，显著降低显存占用

📦

动态分配

按需分配/回收块，避免预分配浪费，支持变长序列，最大序列长度不再受静态缓存限制

⚡

高效内核

定制化 CUDA Kernel 实现块级注意力计算，支持 FP16/BF16/INT8 多种精度，充分利用 Tensor Core
