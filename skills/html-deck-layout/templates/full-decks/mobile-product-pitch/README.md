# mobile-product-pitch

## When to use
产品发布/路演专用模板，适合 SaaS 产品发布、功能升级介绍、融资路演等场景。采用暖橙亲和主题，强调产品价值与商业数据。

## Page breakdown

| # | Title | Layout |
|---|-------|--------|
| 1 | 封面 | Cover (产品名称+一句话价值主张) |
| 2 | 痛点洞察 | Split (左痛点列表+右场景图) |
| 3 | 解决方案 | Card + Diagram (产品架构图) |
| 4 | 核心功能 | 2x2 Grid (四大功能亮点) |
| 5 | 市场数据 | Stat (关键指标+趋势图) |
| 6 | 定价模式 | Split (左定价表+右优势说明) |
| 7 | 产品路线图 | Process Flow (时间线) |

## Customization
- Replace theme: 默认使用 mobile-warm.css，可根据品牌色替换
- Replace content: 修改产品名称、功能点、定价等商务内容
- Add charts: 可嵌入 SVG 图表展示业务数据
