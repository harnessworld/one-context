# mobile-course-intro

## When to use
在线课程/培训介绍专用模板，适合知识付费、技能培训、公开课预告等场景。采用自然/极简主题，强调学习路径和课程价值。

## Page breakdown

| # | Title | Layout |
|---|-------|--------|
| 1 | 封面 | Cover (课程名称+讲师信息) |
| 2 | 课程简介 | Card (一页总结核心内容) |
| 3 | 你将学到什么 | 2x2 Grid (四大模块) |
| 4 | 课程大纲 | Process Flow (学习路径时间线) |
| 5 | 讲师介绍 | Split (左头像+右资历介绍) |

## Customization
- Replace theme: 默认使用 mobile-nature.css，适合教育/知识类主题
- Replace content: 修改课程名称、模块内容、讲师信息
- Adjust images: 添加讲师头像或课程相关示意图
