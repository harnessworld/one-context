# mobile-generic

## When to use
通用型移动演示模板，适用于产品介绍、工作汇报、培训讲解等多种场景。采用简洁的卡片式布局，支持 2x2 网格、左右分栏、图文混排等常见版式。

## Page breakdown

| # | Title | Layout |
|---|-------|--------|
| 1 | 封面 | Cover (居中标题+副标题+标签) |
| 2 | 四大要点 | 2x2 Grid (四宫格卡片) |
| 3 | 左右对比 | Split (左文右图/数据) |
| 4 | 架构图 | Card + Diagram (卡片+SVG图示) |
| 5 | 核心数据 | Stat (超大数字展示) |
| 6 | 感谢页 | Thanks (居中结束语) |

## Customization
- Replace theme: change CSS link in `<head>` (mobile-tech.css, mobile-warm.css, etc.)
- Replace content: edit text and SVG in each slide
- Adjust colors: modify CSS variables in base.css or create new theme file
