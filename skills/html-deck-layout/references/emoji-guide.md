# Emoji 速查表

> 按场景分类的 emoji 速查，用于 HTML 幻灯快速表达。零依赖，移动端原生渲染。

**使用方式**：
```html
<span style="font-size:84px">🏗️</span>
```

**注意**：
- 不同操作系统渲染样式不同（Apple/Google/Microsoft）
- 不可改色，如需统一颜色请用 SVG 图标

## 使用示例

### 卡片标题
```html
<div style="font-size:84px;line-height:1;flex-shrink:0">🏗️</div>
<div style="font-size:52px;font-weight:900">系统架构</div>
```

### 状态标识
```html
<div style="display:flex;align-items:center;gap:10px">
  <span style="font-size:32px">✅</span>
  <span>部署成功</span>
</div>
```

### 进化条
```html
<div style="display:flex;align-items:center;gap:24px">

    <div style="font-size:56px">🐣</div>
```
